## 📍 Installation
```sh
npm i regex-pattern
```
## 🧪 How to use universal
```js

//Import
import {mail} from 'regex-pattern' 

const email = 'example@xample.com'
// Use
const results = mail(email)
console.log(results) // boolean
//or
return results // boolean
```
## 🧪 Result
```js
true 
```
